# 🍽 Spec: F&B Module — CoffeeTree POS

Module này quản lý toàn bộ thực đơn, danh mục và đơn vị tính cho hệ thống F&B.

---

## 1. Danh Mục Sản Phẩm (Category)

Quản lý phân loại món ăn/đồ uống (VD: Cà phê, Trà, Bánh ngọt).

| Attribute | Value |
|---|---|
| `_name` | `categories` |
| `_menu_parent` | `STOCK` |
| `_menu_sequence` | `10` |

### Fields:
- `name` (CharField): Tên danh mục (Unique, Required)
- `image` (ImageField): Ảnh đại diện danh mục

---

## 2. Đơn Vị Tính (Unit)

Quản lý các đơn vị đo lường và quy đổi (VD: Ly, Cái, Thùng).

| Attribute | Value |
|---|---|
| `_name` | `units` |
| `_menu_parent` | `STOCK` |
| `_menu_sequence` | `5` |

### Fields:
- `name` (CharField): Tên đơn vị (Unique, Required)
- `relative_factor` (FloatField): Hệ số quy đổi (Required)
- `relative_uom_id` (ForeignKeyField): Trỏ về chính module `units` để xác định đơn vị gốc.

---

## 3. Sản Phẩm (Product)

Thông tin chi tiết món ăn, đồ uống, giá bán và tồn kho.

| Attribute | Value |
|---|---|
| `_name` | `products` |
| `_menu_parent` | `STOCK` |
| `_menu_sequence` | `20` |
| `_search_fields` | `["name", "sku"]` |
| `_filter_fields` | `["category_id", "is_active"]` |
| `_readonly_fields`| `["sku"]` |

### Fields:
- `name` (CharField): Tên sản phẩm (Required)
- `sku` (CharField): Mã quản lý (Unique, Readonly)
- `price` (DecimalField): Giá bán niêm yết (Required, Precision 12, Scale 2)
- `cost_price` (DecimalField): Giá vốn (Precision 12, Scale 2)
- `category_id` (ForeignKeyField): Liên kết tới `categories` (Required)
- `unit_id` (ForeignKeyField): Đơn vị tính chính (Required, Liên kết tới `units`)
- `unit_ids` (ManyToManyField): Các đơn vị tính phụ (Liên kết tới `units`)
- `is_active` (BooleanField): Trạng thái hoạt động (Default: True)
- `tracking_stock` (BooleanField): Có theo dõi tồn kho hay không (Default: True)
- `image` (ImageField): Ảnh sản phẩm
- `description` (TextField): Mô tả chi tiết món ăn
- `purchase_taxes` (JSONField): Cấu hình thuế mua hàng
- `sale_taxes` (JSONField): Cấu hình thuế bán hàng

---

## 4. Dữ Liệu Mẫu (Seed Data)
Dữ liệu mẫu được lưu trữ tại `backend/seeds/`:
- `categories.json`
- `units.json`
- `products.json`
